# Mock Telco API Capability Map

## REST APIs

### Customer360API

Capabilities:

- Subscriber profile lookup
- Consent retrieval and update
- Eligibility check for 5G SA, eSIM, roaming pass, and premium network products

Telco value:

- Partner onboarding
- Omnichannel care
- Privacy-aware customer APIs
- PII classification and consent enforcement

### NetworkSliceAPI

Capabilities:

- Slice catalog
- Slice reservation
- Cell utilization/status

Telco value:

- 5G network monetization
- Quality-on-demand
- Enterprise private 5G
- SLA-driven partner services

### PartnerChargingAPI

Capabilities:

- Partner settlement
- Usage summary
- Monetization plans

Telco value:

- Usage-based billing
- Revenue attribution
- Partner invoice reconciliation
- Business analytics

## SOAP APIs

### BillingAdjustmentSOAP

Capabilities:

- Create billing credit/debit adjustment
- Return audit-required marker
- Preserve WSDL contract for legacy tooling

Telco value:

- Legacy BSS modernization without immediate backend rewrite
- Governed exposure through API Manager
- SOAP UI / enterprise integration compatibility

## Streaming APIs

### NetworkEventsStreamAPI

Capabilities:

- Network alarm stream
- Slice utilization stream
- Quality-on-demand breach stream
- Roaming partner degradation event
- Charging reconciliation event

Telco value:

- Real-time assurance
- Event-based monetization
- Premium network SLAs
- Partner operational transparency
