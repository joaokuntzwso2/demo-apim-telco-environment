# Open Gateway / CAMARA-style Fraud & Trust Pack

This demo adds a business-oriented Open Gateway / CAMARA-style API product pack for fraud prevention.

## Business scenario

A digital bank, fintech, wallet, marketplace or e-commerce platform wants to reduce fraud during onboarding, checkout, account recovery and high-value transactions.

The partner consumes three operator network API products:

- OpenGatewayNumberVerificationAPI
- OpenGatewaySimSwapRiskAPI
- OpenGatewayDeviceLocationVerificationAPI

## Commercial story

WSO2 APIM publishes, governs, secures and throttles the APIs.

Moesif is used as the usage monetization layer. APIM stores Moesif export fields as API custom properties:

- ConnectedAccountKey
- RevenueShareModel
- SettlementOwner
- ProductLine
- BillingCatalogReference

## Demo message

APIM governs and enforces the commercial API contract.

Moesif meters, rates and monetizes real API usage.
